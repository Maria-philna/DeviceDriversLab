1. kthreadd        - kthreadd is a daemon thread that runs in kernel space. Used to spawn newer 					 threads if required.

2. thermald        - Prevents machine from overheating

3. dbus-daemon     - It is a D-bus message but deamon. 

4. cupsd	       - Scheduler for the Common UNIX Printing System

5. acpid 	       - It provides intelligent power management on a system and allows to query 
				     battery and configuration status by supporting the Advanced Configuration and Power Interface 

6. udisksd         - It serves as an interface to system block devices, implemented via D-Bus. It
				     handles operations such as querying, mounting, unmounting, formatting, or detaching storage devices such as hard disks or USB thumb drives.

7. dockerd         - Listens for Docker API requests and manages Docker objects such as images, 					 containers, networks, and volumes.

8. fwupd           - A simple daemon allowing you to update some devices' firmware, including 
					 UEFI for several machines.

9.systemd-udevd    - Executes matching instructions specified in udev rules.

10.rsyslogd        - Provides a kind of logging that many modern programs use.

11. ModemManager   - It is a DBus-activated daemon which controls mobile broadband (2G/3G/4G) 
				     devices and connections.

12. irqbalance     - Linux daemon that distributes interrupts over among the processors and cores
				     in your computer system.

13. whoopsie       - responsible for collecting error reports from apport and then sending that
				     report to Canonical if the user agrees to this in the apport confirmation dialog.

14. wpa_supplicant - This tool can configure network interfaces and connect to wireless 
					 networks. It is intended to run as a daemon and other command to connect it. 

15. cron     	   - It is a long-running process that executes commands at specific dates and 
					 times. 
			  It can be used to schedule activities, either as one-time events or as recurring tasks.
